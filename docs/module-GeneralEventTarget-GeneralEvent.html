Class: GeneralEvent

GeneralEventTarget~GeneralEvent(publishernon-null, argsopt, nullable)

Every invocated callback function is provided with a general event instance as an argument. This instance is immutable and provides information for the callback function via its properties. Example for such information is the reference to the publisher of the event or arguments provided by the publisher.

Constructor

new GeneralEvent(publishernon-null, argsopt, nullable)

An instance to hold information for a callback function. An instance then will be passed as an argument for an invocation of a callback function. The instance is immutable to prevent bugs caused by callback functions altering it. When one event is fired one instance is used for all invocated callback function.
